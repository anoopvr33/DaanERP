# DaanERP

# 🏨 Hotel ERP System

A modern **Hotel ERP (Enterprise Resource Planning)** application designed to simplify and digitize hotel operations. Built with a clean, responsive interface and scalable architecture, it helps manage daily hotel workflows efficiently while providing a seamless user experience.

---

## ✨ Features

* 🔐 Secure Authentication
* 📊 Interactive Dashboard
* 🛏️ Hotel Booking Management
* 👥 Customer Management
* 💳 Payment Tracking
* 📈 Reports & Analytics
* 🤖 AI-Powered Assistance
* 🔍 Smart Search & Filtering
* 📄 Custom Pagination
* 📱 Fully Responsive Design
* ⚡ Fast REST API Integration
* 🛠️ Complete CRUD Operations

---

| Category             | Technologies                    |

## ⚙️ Getting Started

### 1️⃣ Clone the repository

```bash
git clone <repository-url>
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Configure environment variables

Create a `.env` file and add the required environment variables.

### 4️⃣ Start the development server

```bash
npm run dev
```
